GUI Login Form

Overview
This project is a simple and user-friendly GUI Login Form developed using Python Tkinter. It allows users to enter username and password and checks whether the entered details are correct.

Features
- User input for Username and Password  
- Validation of login details  
- Shows message for:
  - Blank input  
  - Correct login  
  - Incorrect login  

Technologies Used
- Python  
- Tkinter (for GUI design)  

Working
1. User enters Username and Password  
2. If both fields are empty → "Blank Not Allowed"  
3. If Username = *Priyanka* and Password = *admin* → "Login Success"  
4. Otherwise → "Incorrect username and password"  

How to Run
1. Open the file GUI_Login_Form.py  
2. Run the program using Python  
3. Enter login details in the window
